# 🚪The-Last-Door -Escape Room Game

A cinematic browser-based escape room game built using HTML, CSS, and JavaScript.

The player must solve puzzles, decode clues, survive multiple levels, and escape an underground AI-controlled laboratory before total shutdown.

🎮 Features
✅ Core Gameplay
Multi-level escape room gameplay
Randomized puzzle codes
Progressive difficulty system
Interactive puzzle mechanics
Inventory system
Hidden clues
Door opening animation


✨ Visual Features
Cinematic cyberpunk UI
Neon glow effects
Flickering lights
Flashlight mouse effect
Smooth transitions
Typing animation
Glassmorphism panels
Screen shake effect
Animated trophies
🔊 Audio Features
Click sounds
Error sounds
Success sounds
Immersive game feedback


🧩 Levels
Level 1 — Access Room

Find the hidden access code using clues from:

terminal
documents
hidden objects
Level 2 — Security Grid

Decode the laser sequence using:

color-to-number logic
hidden system hints
puzzle solving
Level 3 — AI Override Chamber

Stop the AI and unlock the final escape protocol.
